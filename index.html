Real Estate

Data processing and performance analysis of regression models. Scraping of figaro-immo data from Lyon, Marseille, and Paris. The project involves processing and cleaning the data in Python (text, regex, numeric conversion, and encoding categorical variables). It also includes exploratory analysis, data distribution, and normality analysis. Models used: Ridge, Lasso, Decision Trees, Bagging (Bootstrap aggregation), Voting Regressor, Adaptive Boosting, Gradient Boosting, Stochastic Gradient Boosting, Random Forest. The best-performing model was Gradient Boosting.

ResNet from Scratch

After learning PyTorch, I wanted to challenge myself to build and implement the ResNet architecture from scratch while making it customizable (by changing the expansion/dilation factor of the residual blocks, the size of the block filters, etc). Initially testing the architecture on food data, I decided to do something useful. As an individual with a higher chance of developing skin cancer, I wanted to build a model that would help me decide if I should visit a clinician, in case a mole changed its shape or if a new mole appeared. The project was deployed with Streamlit.

Auto Trader

An automated trading bot that uses patterns found in the Naked Forex book. It includes backtesting functionality to evaluate performance before live trading.
